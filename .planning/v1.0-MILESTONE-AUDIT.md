## Milestone Definition of Done

**Voice-first AI briefing system capable of:**
- Connecting Gmail, Google Calendar, Outlook, and Slack via OAuth
- Precomputing a ranked, LLM-generated morning briefing on a cron schedule

- Approval-gated action execution (email, Slack, calendar)
- End-to-end voice session (TTS/STT with barge-in detection)

**Fix scope:** Small — 2 file changes:
1. `scheduler.py:_build_pipeline_kwargs()` — query `load_profile(session, user_id=1)` and add `preferences` to kwargs
