    ```python
    class STTPipeline:
        """Deepgram Nova-3 WebSocket STT with mic capture and UtteranceEnd turn detection.

            self._transcript_parts: list[str] = []

            """Start mic capture and Deepgram WebSocket streaming."""
    ```

    8. On exit: `await dg_connection.finish()`.
