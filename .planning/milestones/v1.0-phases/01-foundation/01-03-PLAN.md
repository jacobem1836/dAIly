    The function must:
    - Accept `client_id: str`, `client_secret: str`, `scopes: list[str]`, `redirect_uri: str = "http://localhost:8080/callback"`
    - Use `google_auth_oauthlib.flow.Flow.from_client_config` (NOT from_client_secrets_file — we pass config dict, not a file path) with `access_type="offline"` and `prompt="consent"` to ensure refresh_token is always issued (Pitfall 1)

    - Server shutdown: use `threading.Event` — the callback sets the event, a background thread monitors and calls `server.shutdown()` on the uvicorn server
    - Return the credentials object (access_token, refresh_token, expiry)

    - Scopes stored as space-separated string
    - Use async session from session_factory

    GOOGLE_READONLY_SCOPES = [

    1. **src/daily/integrations/google/adapter.py**: Implement two concrete adapters:

    `GmailAdapter(EmailAdapter)`:
    - Constructor accepts a Google API credentials object (already decrypted token)
    - Builds Gmail service using `googleapiclient.discovery.build("gmail", "v1", credentials=creds)`
    - `list_emails(since, page_token)`:
      - Use `service.users().messages().list(userId="me", q=f"after:{since_epoch}", pageToken=page_token)` to get message IDs
      - For each message ID, use `service.users().messages().get(userId="me", id=msg_id, format="metadata", metadataHeaders=["Subject", "From", "To"])` to get metadata
      - Map to `EmailMetadata`: extract subject/sender/recipient from headers, timestamp from `internalDate`, is_unread from `UNREAD` in `labelIds`, labels from `labelIds`
      - Return `EmailPage(emails=[...], next_page_token=response.get("nextPageToken"))`
    - CRITICAL: Use `format="metadata"` — NOT `format="full"`. This ensures no body is returned from the API.

    `GoogleCalendarAdapter(CalendarAdapter)`:
    - Constructor accepts a Google API credentials object
    - Builds Calendar service using `googleapiclient.discovery.build("calendar", "v3", credentials=creds)`
    - `list_events(since, until)`:
      - Use `service.events().list(calendarId="primary", timeMin=since.isoformat()+"Z", timeMax=until.isoformat()+"Z", singleEvents=True, orderBy="startTime")`
      - Map to `CalendarEvent`: event_id from `id`, title from `summary`, start/end from `start.dateTime` or `start.date`, attendees from `attendees[].email`, location from `location`, is_all_day from presence of `start.date` vs `start.dateTime`
      - Return list of CalendarEvent objects

    Note: Both adapters run API calls synchronously via the google-api-python-client (which is sync). Wrap in `asyncio.to_thread()` in the async method to avoid blocking the event loop.
