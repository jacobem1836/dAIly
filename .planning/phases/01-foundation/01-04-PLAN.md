    2. **src/daily/integrations/slack/auth.py**: Implement `run_slack_oauth_flow` following the same localhost callback pattern as Google auth:

    - Define minimum scopes constant:
    ```python
    SLACK_BOT_SCOPES = [
        "channels:read",
        "channels:history",
        "im:read",
        "im:history",
        "users:read",
    ]
    ```

    - Build Slack OAuth authorization URL: `https://slack.com/oauth/v2/authorize?client_id={client_id}&scope={",".join(SLACK_BOT_SCOPES)}&redirect_uri={redirect_uri}`
    - Open browser with `webbrowser.open()`
    - Spin up temporary FastAPI server on `127.0.0.1:8080` with GET `/callback` endpoint
    - On callback: receive `code` query param, exchange for bot token via POST to `https://slack.com/api/oauth.v2.access` with `client_id`, `client_secret`, `code`, `redirect_uri`
    - Extract `access_token` from response JSON (`response["access_token"]`) — this is the bot token
    - Slack bot tokens do not expire (no refresh_token), but store with provider="slack" in integration_tokens

    Implement `store_slack_token(bot_token: str, user_id: int, vault_key: bytes, session_factory)`:
    - Encrypt bot_token using `encrypt_token`
    - Create IntegrationToken row: provider="slack", encrypted_access_token=encrypted_token, encrypted_refresh_token=None, token_expiry=None, scopes=" ".join(SLACK_BOT_SCOPES)

    3. **src/daily/cli.py**: Update `slack()` command to call `run_slack_oauth_flow` and `store_slack_token`. Print success message on completion.
