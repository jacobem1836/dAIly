    1. **src/daily/vault/refresh.py**: Implement `refresh_expiring_tokens`:

    ```python
    async def refresh_expiring_tokens(
        session_factory,
        vault_key: bytes,
        buffer_minutes: int = 15,
    ) -> list[dict]:
        """
        Find tokens expiring within buffer_minutes and refresh them proactively.
        Returns list of refresh results: [{provider, user_id, success, error}]
        
        This function is designed to be called by APScheduler in Phase 2,
        but the logic lives here in the vault module.
        """
    ```

    Implementation:
    - Query IntegrationToken rows where `token_expiry IS NOT NULL AND token_expiry < now() + buffer_minutes`
    - For each expiring token:
      - Decrypt the refresh_token using `decrypt_token`
      - Dispatch to provider-specific refresh function based on `provider` field:
        - "google": Use `google.oauth2.credentials.Credentials` with refresh_token, call `credentials.refresh(google.auth.transport.requests.Request())`
        - "outlook": Use `msal.PublicClientApplication.acquire_token_by_refresh_token(refresh_token, scopes)`
        - "slack": Skip — bot tokens don't expire
      - Encrypt new access_token (and new refresh_token if rotated)
      - Update IntegrationToken row: new encrypted tokens, new token_expiry
      - Catch exceptions per-token: log error, continue with next token
    - Return results list for logging/monitoring

    Also implement individual refresh helpers:
    - `_refresh_google_token(refresh_token: str, client_id: str, client_secret: str) -> dict`
    - `_refresh_microsoft_token(refresh_token: str, client_id: str, tenant_id: str, scopes: list[str]) -> dict`

| Threat ID | Category | Component | Severity | Disposition | Mitigation |

| T-1-17 | Elevation of Privilege | microsoft/auth.py | HIGH | mitigate | Only Mail.Read, Calendars.Read, User.Read, offline_access scopes; no write scopes (Task 1) |
| T-1-18 | Information Disclosure | microsoft/auth.py | HIGH | mitigate | Tokens encrypted before DB write (Task 1) |
