<objective>
Implement three coupled improvements that all touch the conversation turn shape:

- **Improvement 1 — adaptive barge-in safety window (600ms):** replace the
  unconditional `stop_event.set()` in `_on_speech_started` with a deferred timer
  that fires after 600ms unless cancelled.
- **Improvement 2 — backchannel detection:** introduce
  `src/daily/voice/utils.py::_is_backchannel`, plus a `filter_utterance` method on
  `VoiceTurnManager`, and gate barge-in / utterance forwarding on it. Use a
  `_was_tts_active_at_speech_start` flag (set in `_on_speech_started`, cleared in
  `speak()` finally) so the filter does not race against `_tts_active` flipping
  to False before `UtteranceEnd` arrives.
- **Improvement 4 — agent acknowledgement phrases:** speak one of
  `["Got it.", "One sec.", "Sure.", "On it.", "Mmhm."]` between user utterance
  acceptance and `run_session`, skipping for exit/quit, approval sub-loop, and the
  very first turn.

Purpose: The behavioural core of natural conversation — these three changes are
inter-locked because they share fields on `VoiceTurnManager` and the same hook
point in `loop.py`.
Output: barge_in.py rewired with timer + backchannel guard; new utils.py module;
loop.py with acknowledgements; impacted tests updated.
</objective>

    """Utility helpers for the voice loop."""

    from __future__ import annotations

    _BACKCHANNEL_PHRASES: frozenset[str] = frozenset({
        "yeah", "yep", "yup", "yes", "ok", "okay", "right", "got it",
        "uh-huh", "mm-hmm", "mmhm", "sure", "alright", "cool", "go on",
        "continue", "and", "so", "mm", "hmm", "interesting", "i see",
        "ah", "oh",
    })


    def _is_backchannel(text: str) -> bool:
        """Return True when `text` looks like a passive listening token.

        Match criteria (per Phase 17 CONTEXT.md):
          - <= 3 words
          - normalized (strip + lower + strip trailing .,!?) is in _BACKCHANNEL_PHRASES
        """
        if not text:
            return False
        normalized = text.strip().lower().rstrip(".,!?")
        if not normalized:
            return False
        if len(normalized.split()) > 3:
            return False
        return normalized in _BACKCHANNEL_PHRASES
    ```

    def filter_utterance(self, text: str) -> bool:
        """Return True to forward the utterance, False to swallow it.

        A backchannel is swallowed when TTS was active at the moment speech
        started. Cancels the pending barge-in timer so TTS keeps going.
        """
        if self._was_tts_active_at_speech_start and _is_backchannel(text):
            self._pending_barge_in_cancelled = True
            if self._barge_in_timer_task is not None and not self._barge_in_timer_task.done():
                self._barge_in_timer_task.cancel()
            return False
        return True
    ```
